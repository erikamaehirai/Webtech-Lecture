Entity Header Fields

Entity Header Field Name Description Example
Allow Valid methods for a specified resource Allow: GET, HEAD
Content-Encoding To allow the document to be compressed without losing the identity of its underlying media type. Content-Encoding: gzip
Content-Language Languages of the intended audience for the enclosed content Content-Language: mi, en
Content-Length The length of the response body Content-Length: 3495
Content-Location An alternate location for the returned data Content-Location: /index.htm
Content-MD5 Providing an end-to-end message integrity check RContent-MD5 : 8c2d46911f3f5a326455f0ed7a8ed3b3
Content-Range Sent with partial entity Content-Range : bytes 0-499/1234
Content-Type Indicates the media-type of the entity body sent to the recipient Content-Type: text/html; charset=utf-8
Expires Gives the date/time after which the response is considered stale Expires: Mon, 20 Mar 2017 16:00:00 GMT
Last-Modified Indicates the time and date at which the origin in server believes the variant was last modified Last-Modified: Sun, 19 Mar 2017 12:45:26 GMT
