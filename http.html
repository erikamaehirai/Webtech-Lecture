Hypertext Transfer Protocol


Hypertext Transfer Protocol is an application layer communication protocol used to access resources.It works as a request-response protocol between a client and server. It runs on the top of the Internet Protocol Suite (TCP/IP) where port 8O is the default port for TCP and port 443 for HTTPS(over SSL/TLS)

    HTTP Request Methods


    GET

  • It is used to request from the server about the retrieval of the source being identified by the request URl and it is returned as an entity in the message body
  • It is used to transfer a current selected representation of the resource

    • HEAD

  • Identical to GET, except the message body is not included in the response
  • It is used to retrieve metadata about the entity implied by request without transferring the entity itself.

    • POST

  • Request that the origin server accept the entity enclosed in the request as a new subordinate of the resource identified by the Request-URI in the Request-Line.
  • It is more appropriate than Get Method
  • It perform resource - specific processing of the entities enclosed in the message body

    • OPTIONS

  • Request for information about the communication options available on the request/response chain identified by the Request-URI.

    • TRACE

  • Request a loop back of the request message
  • i.e. request the server to “echo” back to the client the received request
  • Typically used for testing/diagnostics of the request

    • PUT

  • Request the server to store the enclosed entity in the message under the specified request URI

    • DELETE

  • Request the server to delete the resource identified by the request URI

    • CONNECT

  • Reserved for use of tunneling proxy servers

    HTTP Status Codes


    1xx (informational)

    Status Codes Name Description
    100 continue a server will check the headers of the request
    101 switching protocols a request using protocol negotiation for switching

    2xx (success)

    Status Codes Name Description
    200 ok successful http request
    201 created request has been fulfilled
    202 accepted the process has not been completed but it is accepted
    203 non authoritative information not definitive
    204 no content it will not return any content but it is processed successfully
    205 reset content (submit data via POST) requires for a reset of the document view
    206 partial content will return content that are not complete

    3xx (redirection)

    Status Codes Name Description
    300 multiple choices giving a request with several versions
    301 moved permanently it is no longer in the request
    302 found requires a client to perform a temporary redirect
    303 see other uses the GET method to find the request under another URl
    304 not modified conditional but has missing
    305 use proxy does not fetch it directly
    306 switch proxy it is no longer used
    307 temporary redirect repeated using a url but does not allow http to change

    4xx (client error)

    Status Codes Name Description
    400 bad request the server will not process the request due to an error of the client
    401 unauthorized (protected) used when authentication is required and has failed or has not yet been provided
    402 payment required reserved for future use
    403 forbidden not allowed to access
    404 not found not correct request
    405 method not allowed a request method is not supported for the requested resource
    406 not acceptable requested resource is generating only the content
    407 proxy authentication required the client must first authenticate with the proxy
    408 request timeout server timed out while waiting for the request
    409 conflict request could not be processed because of the conflict
    410 gone resource requested will not be available again
    411 length required request did not specify the length of its content
    412 precondition failed server does not meet one of the preconditions
    413 request entity too large request is larger than the server is able to process
    414 request URI too large URI provided was too long and the server is not able to process it
    415 unsupported media type request entity has a media type which the resource does not support
    416 requested range not satisfiable the client has asked for a portion of the file but the server is not capable of supplying that portion
    417 expectation failed the request was not met by the server
