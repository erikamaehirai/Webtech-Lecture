  8. Date
    9. lets you work dates and times
    implemented in JavaScript 1.0, NES 2.0 and JavaScript 1.1: added prototype property and JavaScript 1.3: removed platform dependencies to provide a uniform behavior across platforms; added ms_num parameter to Date constructor; added getFullYear, setFullYear, getMilliseconds, setMilliseconds, toSource, and UTC methods (such as getUTCDate and setUTCDate).
    Parameters
    milliseconds
    Integer value representing the number of milliseconds since 1
    January 1970 00:00:00.
    dateString
    String value representing a date. The string should be in a format recognized by the Date.parse method.
    yr_num, mo_num, day_num
    Integer values representing part of a date. As an integer value, the month is represented by 0 to 11 with 0=January and 11=December.
    hr_num, min_num, sec_num, ms_num
    Integer values representing part of a date.

  10. Event
    11. The event object contains properties that describe a JavaScript event, and is passed as an argument to an event handler when the event occurs.
  11. FileUpload
    12. A file upload element on an HTML form. A file upload element lets the user supply a file as input.
  12. Form
    13. Lets users input text and make choices from Form elements such as checkboxes, radio buttons, and selection lists. You can also use a form to post data to a server.
  13. Frame
    14. The Frame object is provided a convenience for referring to the objects that constitute frames. However, JavaScript actually represents a frame using a window object. Every Frame object is a window object, and has all the methods and properties of a window object. However, a window that is a frame differs slightly from a top-level window.
  14. Function
    15. Specifies a string of JavaScript code to be compiled as a function.
  15. Hidden
    16. A Text object that is suppressed from form display on an HTML form. A Hidden object is used for passing name/value pairs when a form submits.
  16. History
    17. Contains an array of information on the URLs that the client has visited within a window. This information is stored in a history list and is accessible through the browser’s Go menu.
  17. Image
    18. An image on an HTML form.
  18. Layer
    19. Corresponds to a layer in an HTML page and provides a means for manipulating that layer.
  19. Link
    20. A piece of text, an image, or an area of an image identified as a hypertext link. When the user clicks the link text, image, or area, the link hypertext reference is loaded into its target window. Area objects are a type of Link object.
  20. Location
    21. Contains information on the current URL.
  21. Math
    22. A built-in object that has properties and methods for mathematical constants and functions. For example, the Math object’s PI property has the value of pi.
  22. Navigator
    23. Contains information about the version of Navigator in use.
  23. Number
    24. Lets you work with numeric values. The Number object is an object wrapper for primitive numeric values.
  24. Object
    25. Object is the primitive JavaScript object type. All JavaScript objects are descended from Object. That is, all JavaScript objects have the methods defined for Object.
  25. Option
    26. An option in a selection list.
  26. Password
    27. A text field on an HTML form that conceals its value by displaying asterisks (*). When the user enters text into the field, asterisks (*) hide entries from view.
  27. Plugin
    28. A plug-in module installed on the client.
  28. Radio
    29. An individual radio button in a set of radio buttons on an HTML form. The user can use a set of radio buttons to choose one item from a list.
  29. Reset
    30. A reset button on an HTML form. A reset button resets all elements in a form to their defaults.
  30. Select
    31. A selection list on an HTML form. The user can choose one or more items from a selection list, depending on how the list was created.
  31. String
    32. An object representing a series of characters in a string.
  32. Style
    33. An object that specifies the style of HTML elements.
  33. Submit
    34. A submit button on an HTML form. A submit button causes a form to be submitted.
  34. Text
    35. A text input field on an HTML form. The user can enter a word, phrase, or series of numbers in a text field.
  35. Window
    36. Represents a browser window or frame. This is the top-level object for each document, Location, and History object group.
